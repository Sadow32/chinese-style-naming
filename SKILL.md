# 中式游戏角色起名

## 触发条件

- 用户要求起名、取名、命名
- 语境为游戏角色、小说角色、二次元角色
- 关键词包含：中国风、古风、仙侠、武侠、原神风、鸣潮风、仙剑风

## 名字格式

本 skill 支持三种格式，按需选用：

| 格式 | 字数 | 结构 | 示例 |
|------|------|------|------|
| 双字名 | 2 字 | 无姓氏分隔 | 清宵、鉴心、丹瑾 |
| 姓+名 | 3 字 | 单姓 + 双字名 | 韩菱纱、徐长卿、柳梦璃 |
| 复姓+名 | 3-4 字 | 复姓 + 单/双字名 | 慕容紫英、上官XX |

## 姓氏字库

用于三字名（姓+名）格式。选字标准：常见中文姓氏 + 古风气质。

### 单姓
韩、柳、唐、徐、沈、苏、林、萧、叶、陆、谢、温、裴、江、顾、秦、卫、薛、慕容（慕容已归入复姓）

### 复姓
慕容、上官、欧阳、令狐、独孤、东方、南宫、夏侯

> 复姓后接单字或双字名均可。例：慕容紫英（双字）、独孤月（单字）。

## 组合模板

### 双字名
1. **形+名** — 形容词修饰名词。例：清宵、甘雨、长离
2. **名+名** — 两个意象拼接。例：云堇、丹瑾、景霄
3. **动+名** — 动作+对象。例：凝光、行秋、鉴心
4. **名+动/形** — 名词+状态。例：景燃、霜凝、天河
5. **叠字** — 重复一个雅字。例：秧秧 → 可扩展：絮絮、渺渺、皎皎
6. **姓氏感首字** — 首字像姓氏，次字为意象。例：钟离、胡桃、申鹤

### 三字名（单姓+双字名）
- 姓氏固定 + 双字名按上述双字名模板生成
- 例：韩（姓）+ 菱纱 = 韩菱纱；徐（姓）+ 长卿 = 徐长卿
- 注意姓与名的声调搭配要顺口

### 复姓+名
- 复姓 + 单字名：慕容 X、上官 X
- 复姓 + 双字名：慕容紫英、令狐 X X

## 生成流程

1. 先确认格式偏好：双字名 / 三字名 / 复姓 / 不限。用户未指定则优先出双字名。
2. 确认性别偏好（男/女/均可）。
3. 确认气质方向：清冷、温润、凌厉、端庄、活泼、飘逸、侠气……如用户未指定，从多个方向各出一个。
4. 从字库抽取组合，每个候选配简短意境解析（4-8 字即可）。
5. 每次输出 8-12 个候选，避免一次过多。
6. 用表格呈现：名字 | 格式 | 解析 | 气质

## 平仄规则

命名时用普通话声调判断平仄，规则如下：

| 声调 | 名称 | 平仄 | 示例 |
|------|------|------|------|
| 第一声 ¯ | 阴平 | 平 | 清(qīng)、天(tiān)、春(chūn) |
| 第二声 ˊ | 阳平 | 平 | 云(yún)、长(cháng)、明(míng) |
| 第三声 ˇ | 上声 | 仄 | 武(wǔ)、雪(xuě)、晓(xiǎo) |
| 第四声 ˋ | 去声 | 仄 | 鉴(jiàn)、渡(dù)、静(jìng) |
| 轻声 ˙ | — | 不计入 | 的、了、子 |

> 中古汉语有入声字（如：竹、菊、白、石、月），现代普通话中已散入四声，本 skill 按普通话实际读音判平仄，不追古音。

### 双字名推荐组合

| 组合 | 节奏感 | 示例 | 推荐度 |
|------|--------|------|--------|
| 仄平 | 先抑后扬，最有起伏 | 鉴心、渡川、景燃 | ★★★ 首选 |
| 平仄 | 先扬后抑，收束有力 | 丹瑾、清瑟、霜序 | ★★☆ 良好 |
| 平平 | 舒缓平和，偏温润 | 清宵、甘雨、长离 | ★★☆ 可用，忌三个以上平声连读 |
| 仄仄 | 短促刚硬 | 素锦、皓月 | ★☆☆ 少用，容易拗口 |

> 叠字名（秧秧、絮絮）不受此限，叠字自带节奏，平仄可通融。

### 三字名（姓+名）推荐组合

| 组合 | 示例 | 说明 |
|------|------|------|
| 仄平平 | 武凌霄、沈寒溪 | 最推荐，姓定调+名舒展 |
| 平仄平 | 萧惊鸿、林若云 | 良好，中间顿挫 |
| 平平仄 | 苏听雪 | 柔和收束 |
| 平仄仄 | 韩若锦 | 可用，较重 |
| 仄平仄 | 陆行远 | 可用 |
| 平平平 | — | 禁止，太平淡 |
| 仄仄仄 | — | 禁止，太拗口 |

> 实用口诀：「仄平开头最精神，平平收尾最舒展。宁可平仄交替，不要三声同调。」

### 样本平仄验证

用户精选的 26 个名字，按平仄归类：

**双字名（原神+鸣潮+自补，16个）**：
- 仄平（7）：鉴心、景燃、凝光、行秋、*长离、*甘雨——占比最高
- 平仄（5）：丹瑾、清宵、*云堇、*钟离、*香菱
- 平平（3）：*今汐（jīn xī）、*申鹤（shēn hè，鹤=去声→仄，修正）
- 修正：申鹤 shēn hè = 平仄；今汐 jīn xī = 平平；凌阳 líng yáng = 平平；秧秧 yāng yāng = 平平（叠字豁免）；胡桃 hú táo = 平平

> 实际分布验证：「仄平」为核心偏好，26 个样本中无任何仄仄组合。

### 快速检查法

生成名字后执行三步检查：
1. 标注每个字的声调和平仄
2. 检查有无三连平或三连仄（双字名看整体节奏即可）
3. 朗读一遍，舌头不打结即为通过
